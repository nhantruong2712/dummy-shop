# DummyShop

Simple product browser with favorites functionality. Built with Angular 20.

## Features

- Login system with JWT
- Get products
- Add/remove favorites

## Tech Stack

- Angular 20
- NgRx for state management
- Angular Material
- TypeScript
- Node 22

## Getting Started

```bash
npm install --legacy-peer-deps
npm start
```

Open http://localhost:4200
